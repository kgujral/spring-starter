Spring MVC Starter Project
==============================

No web.xml instead using WebApplicationInitializer and pure JavaConfig.

Requirements
------------
* [Java Platform (JDK) 7](http://www.oracle.com/technetwork/java/javase/downloads/index.html)
* [Apache Maven 3.x](http://maven.apache.org/)

Quick start
-----------
1. `mvn jetty:run`
2. Point your browser to [http://localhost:8090/](http://localhost:8090/)
